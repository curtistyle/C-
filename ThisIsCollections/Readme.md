# Collections List<T>

| method property | description |
|                 |             |
| `Add` | Agrega un elemento al final de la lista |
| `AddRange` | Agrega los elementos de una coleccion al final de la lista |
| `Capacity` | Devuelve la capicidad actual de la lista, es decir, el numero total de elementos que la lista puede sin tener que redimensionarse internamente |
| `Clear` |  Elimina todo los elementos de la lista |
| `Contains` | Determina si el elemento esta en la lista |
| `Count` | Devuelve el numero de elementos de la lista |
| `IndexOf` | Devuelve el indice de la primera aparición de un elemento en la lista |
| `Insert` | Inserta un elemento en la lista en el indice especificado |
| `Remove` | Elimina la primera aparición de un elemento de la lista |
| `RemoveAt` | Elimina el elemento en el indice especificado de la lista |
| `Sort` | Ordena los elementos de la lista |
| `TrimExcess` | Optimiza la capacidad de la lista actual para que coincida con el numero de elementos que contiene la lista |
| `Find` | Busca en la lista el primer elemento que cumple con el predicado especificado |
| `Exists` | Determina si algún elemento de la lista cumple con el predicado especificado |
| `CopyTo` | Copia los elementos de la lista en una matriz unidimensional, comenzando con un indice particular de la matriz |
| `BinarySearch` | Realiza una busqueda binaria en la lista ordenada, devuelve el indice del elemento encontrado |ç
| `ForEach` | Ejecuta una accion en cada elemento de lista `myList.ForEach(x => Console.WriteLine(x));` |
| `Reverse` | Invierte el orden de los elementos de la lista |
